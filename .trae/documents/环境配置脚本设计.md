# 环境配置脚本设计

## 1. Linux 环境配置脚本（`setup_linux.sh`）

### 功能：
- 检查Python和pip是否安装
- 检查并创建虚拟环境
- 安装系统依赖（chromium-browser）
- 安装Python依赖
- 确保不重复安装

### 核心步骤：
1. 检查Python 3是否安装
2. 检查pip是否安装
3. 检查虚拟环境是否存在，不存在则创建
4. 激活虚拟环境
5. 安装chromium-browser（如果未安装）
6. 安装requirements.txt中的依赖（已安装则跳过）
7. 显示环境设置完成信息

## 2. Windows 环境配置脚本（`setup_windows.bat`）

### 功能：
- 检查Python和pip是否安装
- 检查并创建虚拟环境
- 安装系统依赖（如有需要）
- 安装Python依赖
- 确保不重复安装

### 核心步骤：
1. 检查Python 3是否安装
2. 检查pip是否安装
3. 检查虚拟环境是否存在，不存在则创建
4. 激活虚拟环境
5. 安装requirements.txt中的依赖（已安装则跳过）
6. 显示环境设置完成信息

## 3. 脚本特点：
- 自动检查环境，避免重复安装
- 详细的日志输出，方便调试
- 支持虚拟环境管理
- 包含所有必要的依赖安装
- 脚本运行结束后可直接使用

## 4. 脚本使用方法：
- Linux：`chmod +x setup_linux.sh && ./setup_linux.sh`
- Windows：双击`setup_windows.bat`或在命令提示符中运行

## 5. 预期效果：
脚本运行完成后，项目环境将完全配置好，可以直接运行应用程序，无需额外操作。
